# Java 애플리케이션 실행 과정

자바 컴파일러 소스 코드(.java)를 바이트 코드(.class)로 변환한다.

Class Loader가 변환된 바이트 코드를 Runtime Data Area에 로드 시키고,

로딩된 바이트 코드가 실행 엔진(Execution Engine)에 의해서 실행되게 된다.
